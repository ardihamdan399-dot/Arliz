## 🎮 Tentang Game

**Arliz Gameplay** adalah game quiz interaktif berbasis web dengan tema militer/taktis yang menampilkan **50 soal berkualitas** tentang:

- 🏛️ **Sejarah & Filologi** - Peradaban, naskah kuno, dinasti
- 🧮 **Matematika** - Aljabar, persentase, perhitungan
- ⚛️ **Fisika & Sains** - Mekanika, akustik, astronomi
- 🕌 **Sosiologi Islam** - Teologi, budaya, linguistik
- 🌍 **Budaya Global** - Sertifikasi halal, fenomena sosial

## ✨ Fitur Utama

✅ **50 Soal Berkualitas** - Konten edukatif yang menantang  
✅ **Sistem Poin Dinamis** - +10 poin per jawaban benar  
✅ **Lives System** - 3 nyawa (setiap salah kehilangan 1 nyawa)  
✅ **UI/UX Modern** - Dark mode premium dengan Tailwind CSS  
✅ **Randomized Questions** - Urutan soal teracak setiap permainan  
✅ **Offline Support** - PWA dengan Service Worker caching  
✅ **Responsive Design** - Compatible dengan desktop, tablet, mobile  
✅ **Real-time Score Tracking** - Dashboard interaktif  
✅ **Mode Indicator** - Status Online/Offline terlihat jelas  
✅ **Installable** - Install sebagai native app di Android/iOS  

## 🎯 Cara Bermain

1. **Buka Game** - Klik "MULAI OPERASI 🚀"
2. **Baca Pertanyaan** - Pahami soal dengan baik
3. **Buka Jawaban** - Klik "Buka Kunci Jawaban"
4. **Pilih Verifikasi** - "BENAR" atau "SALAH"
5. **Kumpulkan Poin** - Benar = +10, Salah = -1 Nyawa
6. **Selesaikan 50 Soal** - Raih kemenangan!

### Mekanisme Poin

| Event | Poin | Effect |
|-------|------|--------|
| Jawaban Benar | +10 | Score naik |
| Jawaban Salah | -1 Nyawa | Nyawa berkurang |
| Kehilangan 3 Nyawa | Game Over | Misi Gagal |
| Selesai 50 Soal | Total Score | Misi Sukses |
